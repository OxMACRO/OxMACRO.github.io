Apex Legends Lua Script

Features

  • Laser sight support for SMGs to allow you fire without aiming.
  • All weapons are calibrated to their highest mag capacity, but Rampart bonus excluded.
  • Configurable sensitivity 0.5 - 7.0
  • Indicator that uses lock keys light to give you information with fully configurable interval and buttons. (Disabled, Switch, CurrentIndex)
  • Dynamic key combinations that will show up on console log to help your initial weapon switch.
  • Both types of burst weapons are supported.

Full Package Features

  • Robotic, Humanizer and Randomizer support.
  • All weapons supported both with and without hop-ups such as Double-Tap trigger, Turbocharger.
  • Autofire semi automatic weapons such as G7 Scout, P2020 and burst weapons.
  • Narrator that tells you which weapon you've switched to.
  • Completely configurable Index system that divides weapons to 6 per Index that you can switch with either g keys/mouse or modifier keys (shift, ctrl, alt).
  • Secondary weapon switch key to allow you using same modifier for 2 different weapons.
  • A dedicated switch key that will switch between 2 choosen weapons with one button.
